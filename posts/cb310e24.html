Arduino IDE安装

  • Windows
  • Mac OS X
  • Linux:
    • 32 bits
    • 64 bits
    • ARM
    • ARM 64

ESP8266开发环境配置

1. 添加包索引网址

文件-首选项-设置-附加开发板管理器网址:

2. 安装ESP8266开发板组件库

工具-开发板-开发板管理器:
  搜索esp8266并安装最新版本

3. 选择开发板型号

工具-开发板-ESP8266 Boards:
  选择合适的型号(如:NodeMCU 1.0)

4. 附加库安装

  • 从GitHub等相关位置获取开发板附加库
    • 将文件解压存放在项目文件夹对应位置(可在文件-首选项-项目文件夹位置查看目标位置, 默认值为用户文档-Arduino文件夹)
    • 项目-加载库-添加.ZIP库位置直接从压缩文件安装附加库
  • 从项目-加载库-管理库位置搜索并安装第三方库

环境验证

  文件-示例-ESP8266-Blink打开示例程序, 编译并上传至开发板, 观察板载LED灯闪烁效果。
